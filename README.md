# **How to use our program:**
- Run the .py file in your compiler of choice
- Start the program
- Look at the list of locations and type one out to find the fastest, lowest-risk path*
- Continue trying different locations or press q then enter to exit the program

*Be sure to acknowledge the events that are taking place, as they can affect the risk levels between locations
